OpenAI: структурированный tool routing в API OpenAI structured tool-routing in API release

OpenAI добавила в API слой структурированного tool routing: разработчики описывают политики маршрутизации, которые выбирают между зарегистрированными инструментами до начала генерации, вместо того чтобы полагаться на свободный выбор модели. Первые тестеры отмечают заметно меньше ошибочно маршрутизированных вызовов в агентах с несколькими инструментами. Фича в бете, включается флагом на уровне запроса.

OpenAI added a structured tool-routing layer to its API: developers declare routing policies that pick between registered tools before generation starts, instead of relying on the model's free-form choice. Early testers report noticeably fewer mis-routed calls in multi-tool agents. The feature is in beta behind a per-request flag.

Опубликовано руководство по применению EU AI Act EU AI Act enforcement guidance published policy

Еврокомиссия опубликовала проект руководства по применению AI Act в части обязательств для general-purpose моделей: уточнены пороги по документации и переходные сроки для моделей, уже выведенных на рынок. Национальные регуляторы получают единый шаблон отчётности об инцидентах. Публичные комментарии принимаются до конца июля.

The European Commission published draft enforcement guidance for the AI Act's general-purpose model obligations, clarifying documentation thresholds and grace periods for models already on the market. National regulators get a shared incident-reporting template. The public comment window runs through late July.

Видеогенерация: согласованность на длинном контексте растёт Video gen: long-context consistency improving trend

Несколько лабораторий показали минутные сгенерированные ролики со стабильными персонажами и геометрией сцены. Общий приём — memory-augmented diffusion: генератор опирается на постоянное состояние сцены, а не только на последние несколько кадров. После отметки в 90 секунд качество всё ещё заметно деградирует.

Several labs demoed minute-long generated clips with stable characters and scene geometry. The common thread is memory-augmented diffusion: the generator conditions on a persistent scene state rather than only the last few frames. Quality still degrades noticeably past the 90-second mark.

https://example.com/videogen-consistency
